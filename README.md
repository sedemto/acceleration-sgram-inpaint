# Acceleration of spectrogram inpainting
Supplementary material for the research paper *Acceleration of Spectrogram Inpainting by Support Restriction* submitted to the STUDENT EEICT 2026 conference.

# Contents
- The algorithm for optimal support restriction is under the script name `opt_sgram_supp.m`.
- The simple inpainting method is contained within the function `inpaint.m`.
- The three demos mentioned in the paper.
- The algorithm for minimum (suboptimal) support restriction `opt_sgram_supp.m`.  

# Dependencies
The Matlab codes for U-PHAIN-TF use the [LTFAT](https://ltfat.org/) and the Signal Processing Toolbox. We used Matlab R2025a in our experiments.
