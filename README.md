# Acceleration of spectrogram inpainting
Supplementary material for the STUDENT EEICT 2026 conference containing MATLAB scripts.

The algorithm for optimal restriction is under the script name `opt_sgram_supp.m'.

The Matlab codes for U-PHAIN-TF use the [LTFAT](https://ltfat.org/) and the Signal Processing Toolbox. We used Matlab R2025a in our experiments.
